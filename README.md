# Neighbourhood Reporter

A web application that allows citizens to report real-life neighborhood issues such as potholes, garbage dumps, water leaks, broken streetlights, and more. Reports appear on an interactive map, and authorities/admins can update status, add notes, and resolve issues.

## MVP Features
- Report issues with photos, description, category, location
- Auto-detect GPS location or manual pin on map
- View issues on an interactive map
- Issue status tracking (Open → In Progress → Resolved)
- User dashboard to view own reports
- Admin dashboard for managing reports

## Tech Stack
- **Frontend:** Next.js, React, Tailwind CSS, Leaflet/Mapbox
- **Backend:** Node.js (Express/Fastify), PostgreSQL + PostGIS
- **Storage:** Cloudinary / AWS S3
- **Auth:** JWT
- **Notifications:** Email (SendGrid) or Push (Firebase)

## Project Structure
- `/server` – backend API
- `/web` – frontend (Next.js)
- `/docs` – design docs and architecture diagrams
